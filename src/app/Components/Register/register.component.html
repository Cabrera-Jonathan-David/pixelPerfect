Registro de Usuario

Información de cuenta

El nombre de usuario es obligatorio. Debe tener al menos 5 caracteres. No puede superar los 20 caracteres. Solo letras, números y guiones bajos. El nombre de usuario ya existe.
El correo electronico es obligatorio. El correo debe tener un formato válido. El correo electrónico ya está en uso.
La contraseña es obligatoria. Debe tener al menos 8 caracteres. Debe incluir una letra mayúscula, un número y un símbolo especial.
Las contraseñas no coinciden.

Información personal

El nombre es obligatorio. Debe tener al menos 2 caracteres. No puede superar los 50 caracteres. Solo letras y espacios permitidos.
El Apellido es obligatorio. Debe tener al menos 2 caracteres. No puede superar los 50 caracteres. Solo letras y espacios permitidos.
El DNI es obligatorio. Solo se permiten números. El DNI ya existe.
La fecha es obligatoria. Debes ser mayor de 18 años.

Información de contacto

El codigo de área es obligatorio. Solo se permiten números.
El telefono es obligatorio. Solo se permiten números.

Información de dirección

La dirección es obligatoria.
La altura es obligatoria. Solo se permiten números.
El campo es obligatorio.
La ciudad es obligatoria.
La provincia es obligatoria.
El pais es obligatorio.
